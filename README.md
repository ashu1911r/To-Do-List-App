# To-Do List App

A simple command-line To-Do List application built using Python.

## Features

- Add Tasks
- View Tasks
- Mark Tasks as Completed
- Delete Tasks
- Save Tasks Permanently using JSON

## Technologies Used

- Python
- JSON
- File Handling

## How to Run

1. Open terminal
2. Navigate to project folder
3. Run:

```bash
python todo.py
```
